Dependency Libraries

External Libraries Used by ClanLib 3.0 Windows

ClanLib 3.0 does not require external libraries. However some examples use them.

  • MikMod, required by the mikmod example.
  • assimp, required by the 3D examples.
  • fribidi, required by the language example.

External Libraries Used by ClanLib 2.3 Windows

  • zlib, required by clanCore
  • libjpeg, required by clanDisplay
  • libpng, required by clanDisplay
  • FreeType, required by clanDisplay
  • MikMod, required by clanMikMod
  • libOgg, required by clanVorbis
  • libVorbis, required by clanVorbis
  • pcre, required by clanRegExp
  • sqlite, required by clanSqlite
  • assimp, required by the 3D examples. (Not required for ClanLib SDK)
  • fribidi, required by the language example. (Not required for ClanLib SDK)
